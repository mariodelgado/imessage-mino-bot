# Mino Multi-Platform Architecture Plan

## Executive Summary

Build a unified AI assistant platform supporting:
1. **iMessage Bot** - Current implementation, enhanced with Claude handoff
2. **React Native iOS App** - On-device LLM + cloud hybrid with streaming Mino sessions
3. **Shared Backend** - WebSocket server with ADK bidi-streaming

## Summary

This architecture provides:

1. **Unified Experience** - Same AI assistant across iMessage and native app
2. **Real-time Streaming** - ADK bidi-streaming for live browser sessions
3. **Offline Capable** - On-device LLM for quick responses
4. **Rich Artifacts** - Structured data storage and display
5. **Morning Updates** - Scheduled Mino calls with push notifications
6. **Claude Integration** - Seamless handoff for complex tasks
